#  Little Lemon Booking Form

A simple, responsive web form that allows customers to book a table at the **Little Lemon restaurant**.

---

## Features
✅ Collects customer details (name, email, phone, date, time, guests)  
✅ Validates inputs before submission  
✅ Shows confirmation message  
✅ Responsive and user-friendly  

---

## Built With
- HTML5  
- CSS3  
- JavaScript
